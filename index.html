Understanding typeof Operator

What is typeof?

The typeof operator returns a string indicating the type of the unevaluated operand.

Syntax: typeof operand
or: typeof(operand)

Possible Return Values

  • "undefined" - for undefined values
  • "boolean" - for Boolean values
  • "number" - for numbers (including NaN and Infinity)
  • "string" - for strings
  • "symbol" - for symbols (ES6+)
  • "bigint" - for BigInt values (ES2020+)
  • "function" - for functions
  • "object" - for objects, arrays, and null

Important Notes

  • typeof null returns "object" (this is a historical bug in JavaScript)
  • typeof NaN returns "number" (NaN is technically a number)
  • Arrays are objects, so typeof [] returns "object"
  • Use Array.isArray() to check if a value is an array
